너에게의 기능

따뜻한 기술로, 소중한 일상을 지킵니다

☑

일일 체크인

매일 기분을 선택하고 체크인 버튼 하나만 누르세요. "오늘도 잘 있다"는 안부가 가족에게 전해집니다.

  • • 3가지 기분 선택 (좋음/보통/힘듦)
  • • 한 번의 터치로 완료
  • • 주간 체크인 기록 확인
💬

AI 대화 일기

AI(너에게)가 "오늘 하루 어떠셨어요?"라고 먼저 말을 걸어요. 편하게 대화하면 일기가 자동으로 완성됩니다.

  • • 따뜻하고 공감하는 대화
  • • 대화 → 일기 자동 정리
  • • 지난 일기 다시 보기
🛡

지킴이 알림

하루 동안 체크인이 없으면, 등록한 지킴이(가족, 이웃)에게 자동으로 안부 확인 알림이 전송됩니다.

  • • SMS, 이메일 자동 알림
  • • 지킴이 최대 2명 (무료)
  • • 알림 시간 직접 설정
💌

레거시 메시지

소중한 사람에게 전하고 싶은 마음을 미리 기록해두세요. PIN 잠금으로 안전하게 보관하고, 원하는 때에 전달할 수 있습니다.

  • • 지킴이별 개별 메시지 (프리미엄)
  • • 텍스트 편지 작성
  • • PIN 잠금으로 안전 보관

이렇게 간단합니다

3단계로 시작하세요

1

매일 체크인

앱을 열고, 오늘의 기분을 선택하고, 체크인 버튼을 누르세요. 10초면 끝납니다. 정해진 시간까지 체크인이 없으면 지킴이에게 알림이 갑니다.

2

AI와 대화로 기록

체크인 후, 오늘 있었던 일을 편하게 이야기하세요. AI(너에게)가 "그랬군요", "그때 어떤 기분이셨어요?"라고 물어봅니다. 대화가 끝나면 일기가 자동으로 정리됩니다.

3

마음을 미리 전하세요

소중한 사람에게 전하고 싶은 말을 레거시 메시지로 남겨두세요. PIN 잠금으로 안전하게 보관되고, 원하는 때에 전달할 수 있습니다.
