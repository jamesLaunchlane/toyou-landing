- **목표**: 사전 신청 (Lead Generation)

- **입찰 전략**: Target CPA (₩5,000)

## Ad Group 1: 노인 안부 확인

**키워드**:
- 부모님 안부 확인
- 독거 노인 안부
- 부모님 안부 앱
- 노인 안부 서비스
- 혼자 사시는 부모님
- 부모님 안전 확인

**반응형 검색 광고**:
| 헤드라인 | 설명 |
|----------|------|
| 부모님 안부, 매일 확인하세요 | 매일 체크인으로 부모님이 잘 계신지 확인. 미체크인 시 가족에게 자동 알림. 사전 신청 무료. |
| 부모님이 괜찮으신지 걱정되세요? | 하루 한 번 체크인이면 충분합니다. 체크인이 없으면 지킴이에게 자동 알림. 지금 사전 신청하세요. |
| 독거 부모님을 위한 안심 앱 | 매일 2분 체크인으로 안부 확인. SMS·이메일 자동 알림. 무료로 시작하세요. |

## Ad Group 2: 시니어 일기 / 기록

**키워드**:
- 노인 일기 앱
- 시니어 기록
- 하루 기록 앱
- 감정 일기
- 대화형 일기
- 어르신 일기

**반응형 검색 광고**:
| 헤드라인 | 설명 |
|----------|------|
| 매일 2분, 대화로 쓰는 일기 | 어려운 글쓰기 없이, 대화만 하면 일기가 완성됩니다. 마음 건강을 지키는 2분 습관. |
| 오늘 하루 어떠셨어요? | 따뜻한 대화로 하루를 기록하세요. 글을 쓸 필요 없이 대화만 하면 일기가 됩니다. |
| 시니어를 위한 감정 일기 앱 | 매일 기분을 기록하고, 대화하듯 하루를 정리하세요. 무료로 시작. |
